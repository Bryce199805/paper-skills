# Paper Tools

This repository provides paper-focused skills for CV/ML/DL research workflows.

Available skills live under `skills/`:

- `paper-read`: read one local paper and produce a structured research note.
- `paper-review`: review one local paper with quick or collaborative peer-review workflows.

When a user asks to read a paper, use `paper-read`. When a user asks to review, referee, score, or draft reviewer comments for a paper, use `paper-review`.

Prefer local paper files over web lookup. Use the skill's bundled parser script when local text extraction is needed.
