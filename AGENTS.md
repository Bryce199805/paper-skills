# Paper Tools

This repository contains portable `SKILL.md` skills for paper workflows.

Core skills:

- `skills/paper-read`: research-oriented reading for one local CV/ML/DL paper.
- `skills/paper-review`: quick or collaborative peer review for one local CV/ML/DL paper.

Agent guidance:

- Load `paper-read` when the task is reading, understanding, summarizing, or creating research notes for a paper.
- Load `paper-review` when the task is reviewing, refereeing, scoring, or drafting reviewer comments for a paper.
- Process one paper at a time.
- Prefer local files. Do not replace local paper text with web content unless the user asks.
- Keep formal reviewer comments in English when using `paper-review`.
